# Sanway Event Talks App

This project provides a single-page website to display the schedule for a technical event filled with talks.
It features a clear, interactive schedule with talk details and a search functionality that allows users to filter talks by category.

## Features

*   **Event Schedule:** Displays a detailed schedule of technical talks, including timings, titles, speakers, categories, and descriptions.
*   **Lunch & Transition Breaks:** Clearly indicates breaks within the schedule.
*   **Category Search:** Allows users to easily find talks of interest by filtering based on their categories (keywords).
*   **Single-Page Application:** All content and functionality are contained within a single HTML file for easy deployment and sharing.
*   **Responsive Design:** Optimized for viewing on various devices, from desktops to mobile phones.

## Technologies Used

*   **Node.js:** Used as a build tool to generate the static `index.html` file.
*   **HTML5:** For structuring the web content.
*   **CSS3:** For styling and responsive design.
*   **JavaScript (ES6+):** For dynamic content rendering and search functionality.

## Setup and Running the Project

To set up and run this project locally, follow these steps:

1.  **Clone the repository:**
    ```bash
    git clone https://github.com/SanwaySarkar/Sanway-event-talks-app.git
    cd Sanway-event-talks-app
    ```

2.  **Generate `index.html`:**
    The `index.html` file, which contains the entire website, is generated by a Node.js script. Ensure you have Node.js installed.
    ```bash
    node generate-website.js
    ```
    This command will create `index.html` in the project root directory.

3.  **Serve the Website Locally:**
    You can use Python's built-in HTTP server to view the website.
    ```bash
    python3 -m http.server
    ```
    (For Python 2, use `python -m SimpleHTTPServer`).

4.  **Access the Website:**
    Open your web browser and navigate to `http://localhost:8000`.

5.  **Test Functionality:**
    Use the search bar at the top of the page to filter talks by categories (e.g., "AI", "Web Development", "Kubernetes").
